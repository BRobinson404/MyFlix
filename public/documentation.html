Documentation Page

INSTRUCTIONS:

This app is designed to navigate you through some of the best movies ever made!

Here you will find information on how to access the API to get the Titles, Directors and genres

You can also register, update your personal info, and create your own list of favorite movies.

ENDPOINT DOCUMENTATION:

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET a JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/Title GET a JSON object holding data about a single movie containing the data above
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/Genre/Genrename GET JSON object holding data about genre
Return data about a director (bio and oscar nominations ) by name /movies/Directors/Directorname GET JSON object holding data about director
Allow new users to register /users POST Returns New User .JSON and logs New User Created
Allow users to update their user info (username) /users/username PUT returns .JSON with updated information and logs Username Updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/username/favoriteMovieID POST a text message that movie was added to favorite list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/username/favoriteMovieID DELETE text message that movie was removed from favorites list
Allow existing users to deregister (showing only a text that a user email has been removed) /users/username DELETE a text message indicating successfull deletion of the user account
